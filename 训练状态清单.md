## 现有可直接使用的文件

### 脚本

| 文件 | 作用 |
|------|------|
| `scripts/train_qwen2vl_lora.py` | LoRA 训练入口（支持 --resume-from 续训、--use-qlora） |
| `scripts/evaluate.py` | LoRA adapter 评测（支持 --base-only 基座对比） |
| `scripts/prepare_chartqa_lora_package.py` | 自动扫描合并所有 JSONL → train/val/test |
| `scripts/prepare_public_datasets.py` | HF 公有数据集导出（docvqa/textvqa/vqa_v2） |
| `scripts/prepare_vqa_v2_local.py` | 本地 VQA-v2 JSON + COCO 图片 → JSONL |
| `scripts/prepare_textvqa_local.py` | 本地 TextVQA JSON + OpenImages 图片 → JSONL |
| `scripts/download_coco_images.py` | COCO 图片并发下载 |
| `scripts/extract_textvqa_images.py` | 从 HF 缓存提取 TextVQA 图片 |
| `scripts/split_train_val.py` | JSONL 切分训练/验证集 |
| `scripts/run_train.sh` | 训练启动脚本 |
| `scripts/run_ablation.sh` | 消融实验一键运行（8 组） |
| `scripts/summarize_ablation.py` | 消融实验结果汇总 + 效应分析 |
| `scripts/evaluate.py` | LoRA adapter 评测（新增 --no-ocr / --no-evidence-prompt 开关） |
